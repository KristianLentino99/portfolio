# Kristian Lentino Brand System

Pi package containing the complete personal brand system for Kristian Lentino — engineer-founder, AI agent builder at commercetools, co-founder of Koomy.

## Install

```bash
pi install ./packages/kristian-lentino-brand
```

Or from git (after pushing):

```bash
pi install git:github.com/kristianlentino/kristian-lentino-brand
```

## What's Included

- **SKILL.md**: Full brand guide with tone of voice, color tokens (OKLCH + hex), typography rules (Space Grotesk, Bangers, monospace), layout system, component vocabulary, motion specs, accessibility rules, and copy/UI checklists.

## Usage

Once installed, the agent auto-loads this skill whenever you work on:
- kristianlentino.it portfolio
- Koomy content or UI
- Any copy, component, or design work that should match Kristian's founder brand
